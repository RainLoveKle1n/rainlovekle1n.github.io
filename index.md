Once you have installed Webra, you can create your desktop applications using Python scripts. Follow these steps to run a simple application:

1. **Open your terminal or command prompt**:
   - For Windows, you can use Command Prompt or PowerShell.
   - For macOS and Linux, use the Terminal.

2. **Create a new project directory**:
   ```
   mkdir my_webra_app
   cd my_webra_app
   ```

3. **Create a Python script**:
   Create a new file named `app.py` and copy the following code into it:
   ```python

   app = WebApp()
   app.load_url("https://example.com")
   app.run()
   ```

4. **Run your application**:
   Enter this command in your terminal:
   ```
   python app.py
   ```

Your application should now open in a window, displaying the website you specified.
